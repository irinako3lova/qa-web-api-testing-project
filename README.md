# Дипломный проект — QA Engineer  

## 📌 Описание  
Итоговый проект в моём портфолио в рамках учёбы в Яндекс Практикум! Это  приложение Яндекс Самокат — сервис, который позволяет арендовать электрический самокат на несколько дней. Я полностью протестировала приложение — и веб, и мобильную версию.
